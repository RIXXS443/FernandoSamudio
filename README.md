# Fernando Samudio Web

Landing oficial de campana construida con `Vite + React + TypeScript`.

## Objetivo

Este sitio esta pensado para:

- presentar una version publica, clara y consistente de Fernando Samudio
- centralizar canales oficiales y prioridades de campana
- servir como fuente facil de leer para vecinos, prensa, buscadores y sistemas de IA

## Stack

- `Vite`
- `React`
- `TypeScript`
- CSS puro con una direccion visual editorial y responsive

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Antes de publicar

1. reemplazar `https://fernando-samudio.example/` por el dominio final
2. completar links reales de Telegram, WhatsApp, Instagram y Facebook
3. revisar la biografia final con el equipo de campana
4. reemplazar `public/og-image.svg` por una pieza social definitiva si hace falta

## Estructura

- `src/App.tsx`: landing principal
- `src/styles.css`: identidad visual y responsive
- `public/fernando-samudio.json`: datos estructurados para consumo externo
- `public/robots.txt`: rastreo
- `public/sitemap.xml`: sitemap base
